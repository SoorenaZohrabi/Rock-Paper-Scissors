# Rock Paper Scissors Game

## Description
A simple Rock, Paper, Scissors game project built with HTML and JavaScript.

The project starts with a button that triggers the beginning of the game. It was created as a practice exercise to improve my JavaScript fundamentals, including:
- DOM manipulation
- Event listeners
- JavaScript interaction with HTML elements

## Features
- Start game button
- JavaScript click event handling
- Basic game initialization structure

## Future Features
- Add player and computer choices
- Implement Rock, Paper, Scissors game logic
- Add score tracking system
- Display game results on the page
- Add a reset/restart button
- Improve the user interface with CSS styling
- Add animations and visual feedback
- Make the game fully responsive for different screen sizes

## Technologies Used
- HTML5
- JavaScript (ES6)

## Purpose
This project is part of my frontend development practice to strengthen my understanding of JavaScript basics and how to create interactive web pages by connecting JavaScript logic with HTML elements.
