# Rock Paper Scissors Game

## Description
A modern and responsive Rock, Paper, Scissors game built with HTML, Sass, Bootstrap, and JavaScript.

This project started as a simple JavaScript practice exercise and was redesigned with a complete UI/UX experience. It focuses on creating an interactive game interface while improving frontend development skills.

The project helps practice:
- HTML structure
- Sass architecture
- Bootstrap responsive layouts
- JavaScript DOM manipulation
- Event handling
- Interactive UI development

## Features
- Responsive design for desktop and mobile devices
- Modern UI built with Bootstrap and Sass
- Custom color palette and styling
- Player choice selection (Rock, Paper, Scissors)
- Computer random choice system
- Score tracking system
- Game result display
- Restart game functionality
- Interactive buttons and hover effects
- Clean component-based Sass structure

## Future Features
- Add animations for winning and losing states
- Add sound effects
- Add game history tracking
- Add difficulty levels
- Add player name customization
- Add local storage for saving scores
- Add dark mode support

## Technologies Used
- HTML5
